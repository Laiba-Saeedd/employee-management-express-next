# Employee Management Express-Next
                                                                                  
A full-stack employee management system using Express.js for backend and Next.js for frontend.

## Tech Stack
- Backend: Node.js, Express.js, MySQL
- Frontend: Next.js, React
- Others: CORS, server.js

## Project Setup

### Backend
1. Navigate to `server/`
2. Install dependencies: `npm install`
3. Create `server.js` file with DB credentials
4. Start server: `node server.js`                      

### Frontend
1. Navigate to `client/`
2. Install dependencies: `npm install`
3. Start Next.js app: `npm run dev`
